# Astrsk Design System

Welcome to the Astrsk Design System documentation.

## Overview

This design system provides a collection of reusable components built with:

- **React 18** - Modern React with hooks
- **TypeScript** - Type-safe component development
- **Tailwind CSS v4** - Utility-first CSS framework
- **Storybook 9** - Component documentation and development
