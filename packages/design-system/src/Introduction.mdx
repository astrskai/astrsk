# Astrsk Design System

A modern React component library.

---

## Tech Stack

- **React 18/19** - Modern React with hooks
- **TypeScript** - Type-safe component development
- **Tailwind CSS v4** - Utility-first CSS framework
- **Storybook 9** - Component documentation

---

## Installation

```bash
npm install @astrsk/design-system
# or
pnpm add @astrsk/design-system
```

---

## Usage

```tsx
// 1. Import styles once in your app's entry point
import '@astrsk/design-system/styles';

// 2. Import components and utilities
import { cn } from '@astrsk/design-system';
```

## Design Principles

### 1. Semantic Tokens

All colors are defined as semantic tokens with automatic dark/light theme support.

```css
/* Primitive (avoid direct use) */
var(--color-neutral-900)

/* Semantic (recommended) */
var(--bg-surface)
var(--fg-default)
var(--border-default)
```

### 2. Theme Support

Dark theme by default. Use `.light` class to switch to light theme:

```html
<html class="light">
  <!-- Light theme -->
</html>
```

### 3. Tailwind Integration

All tokens are also available as Tailwind utilities:

```html
<div class="bg-surface text-fg-default border-border-default">
  ...
</div>
```
