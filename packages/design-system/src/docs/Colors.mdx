## Usage

### CSS Variables

```css
/* Primitive tokens */
background-color: var(--color-neutral-900);
color: var(--color-brand-500);

/* Semantic tokens (theme-aware) */
background-color: var(--bg-surface);
color: var(--fg-default);
border-color: var(--border-default);
```

### Tailwind Classes

```html
<!-- Primitive colors -->
<div class="bg-neutral-900 text-brand-500">...</div>

<!-- Semantic colors -->
<div class="bg-surface text-fg-default border-border-default">...</div>
```
